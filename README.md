项目中使用了 mdui 框架布局，其他就没什么了。

## How to run project

```
git clone https://github.com/orangbus/study-m3u8.git --depth 1
cd study-m3u8
npm i
npm run serve
```

That is ok ,open browser input `http://localhost:8080` ,  then you can test and study .
